html 属性

html 属性提供有关 html 的其他相关信息

html 属性

  • 所有 html 元素都可以有属性。

  • 属性提供元素相关的信息。

  • 属性始终在开始标签上使用。

  • 属性通常用 名称=值 的方式出现，例如：name=”app”。

html 标题

HTML标题是使用<h1>to<h6>标记定义的。

<h1>定义最重要的标题。<h6>定义最不重要的标题。

注意： 浏览器会在标题的前后加上一些边距。在网页上显示一定的空白。

标题很重要

搜索引擎使用标题来索引网页的结构后内容。

用户经常按页面的标题来浏览网页内容，显得变体很重要，<h1> 是主要标题，依次是次要标题 <h2> ,一直到最不重要的标题 <h6>。

注意： 仅将html 标题用于标题，不要为了文本变大加粗而使用标题。 总的来说就是html 元素的使用尽量的使用它的语义化 而不是表面上的样式

链接标签

在HTML中创建超链接非常简单，只需用标签环绕需要被链接的对象即可，其基本语法格式如下：

一些必要属性的说明：

href：用于指定链接目标的url地址，当为标签应用href属性时，它就具有了超链接的功能。 Hypertext Reference的缩写。意思是超文本引用

target：用于指定链接页面的打开方式，其取值有_self 和_blank 两种，其中_self为默认值，_blank为在新窗口中打开方式。
注意：

  1. 外部链接 需要添加 www.baidu.com

  2. 内部链接 直接链接内部页面名称即可比如 < a href="index.html"> 首页 </a >

  3. 如果当时没有确定链接目标时，通常将链接标签的href属性值定义为“#”(即href=”#”)，表示该链接暂时为一个空链接。

  4. 不仅可以创建文本超链接，在网页中各种网页元素，如图像、表格、音频、视频等都可以添加超链接。

锚点定位

通过创建锚点链接，用户能够快速定位到目标内容。
创建锚点链接分为两步：

  1. 使用“a href=”#id名>“链接文本”创建链接文本（被点击的）

  2. 使用相应的id名标注跳转目标的位置。
