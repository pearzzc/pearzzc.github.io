html 简介

html 是一种为创建网页的标准的标记语言

html 是什么？

  • html 代表超文本标记语言。
  • html 是为创建网页的标准的标记语言。
  • html 描述网页结构。
  • html 包含一系列的元素。
  • html元素 告诉浏览器如何展示网页内容。
  • html元素标签也是一系列内容， 例如：标题、段落、超链接，等等。

例子解释：

  • <!DOCTYPE html> 声明和定义这个文件是 html5 的文档

  • <html> 标签元素是 html 网页的 根元素

  • <head> 标签元素包含此网页的元信息，所谓元信息, 即告诉浏览器的一些信息。

  • <title> 为此网页指定一个标题。

  • <bady> 标签定义文档的“身体”，它是一个为了网页中所有可视化内容而生的容器，例如：各级标签，段落，图片，超链接，列表，表格等等。

  • <h1> 定义一个最大的标题

  • <p> 定义一个段落

什么是 html 元素

html 元素主要通过 开始标签，一些内容， 结束标签 ， 三个部分被定义

大部分的 html 元素由开始标签开始 到结束标签结束

大致如下面的代码：

<p>My first paragraph.</p>

开始标签 内容 结束标签
<p> My first paragraph. </p>

当然也有少数的标签是没有结束标签的， 它没有内容比如：<br> 这些元素叫做空元素，空元素没有结束标签。

网页浏览器

流行的网页浏览器（Chrome, Edge, Firefox, Safari）都是为了读取 html 文件并正确的展示内容，浏览器不会展示 html 的标签， 但是他会使用 html 的标签 来确定怎么展示文件。
