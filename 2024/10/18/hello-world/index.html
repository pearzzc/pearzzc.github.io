创建博客

  • 安装 nodejs
  • npm install hexo-theme-butterfly 主题安装
  • npm install hexo-renderer-pug hexo-renderer-stylus --save 插件
  • npm install -g hexo-cli 安装框架
  • hexo init 初始化
  • npm install --save hexo-tag-aplayer 安装音频播放组件
  • npm install --save hexo-deployer-git 上传时安装
  • npm uninstall --save hexo-generator-index ； npm install --save hexo-deployer-git制定时安装

调试博客

  • hexo clean 清理静态内容
  • hexo g 生成静态内容
  • hexo s 本地运行
  • hexo d 静态文件上传github等
  • hexo new title 新增一篇文章
  • hexo new page about/me 新一篇指定路径的文章
