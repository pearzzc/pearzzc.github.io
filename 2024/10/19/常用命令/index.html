常用命令

  • 文件操作命令及其用法示例：

    命令 例子 中文解释说明
    cd cd Documents 进入指定目录
    ls ls -l 列出当前目录下的文件和子目录
    pwd pwd 显示当前目录的路径
    mkdir mkdir myfolder 创建一个新的目录
    rmdir rmdir myfolder 删除一个空目录
    rm rm myfile.txt 删除指定的文件
    cp cp myfile.txt ~/Documents 将指定的文件复制到另一个目录
    mv mv myfile.txt ~/Documents 移动指定的文件到另一个目录
    touch touch myfile.txt 创建一个新的空文件
    cat cat myfile.txt 显示指定文件的内容
    more more myfile.txt 分页显示指定文件的内容
    less less myfile.txt 分页显示指定文件的内容
    head head myfile.txt 显示指定文件的前几行
    tail tail myfile.txt 显示指定文件的后几行
    chmod chmod 755 myfile.txt 更改指定文件的权限
    chown sudo chown root myfile.txt 更改指定文件的所有者
    chgrp sudo chgrp staff myfile.txt 更改指定文件的组
    find find /Users -name myfile.txt 在指定目录中查找指定文件
    grep grep "hello" myfile.txt 在指定文件中查找指定字符串
    open open myfile.txt 使用默认应用程序打开指定文件

  • 补充说明以下是 mv、mkdir、rm 、cp命令的一些常用参数及其用法示例：

    命令 参数 例子 中文解释说明
    mv -v mv -v myfile.txt ~/Documents 移动指定的文件到另一个目录，并显示详细信息
    mv -i mv -i myfile.txt ~/Documents 移动指定的文件到另一个目录时，如果目标目录已存在同名文件，提示用户是否覆盖
    mv -n mv -n myfile.txt ~/Documents 移动指定的文件到另一个目录时，如果目标目录已存在同名文件，不覆盖该文件
    mkdir -p mkdir -p myfolder/subfolder 创建一个新的目录，如果该目录的上级目录不存在，则同时创建上级目录
    rm -f rm -f myfile.txt 强制删除指定的文件，不提示用户
    rm -r rm -r myfolder 递归删除指定的目录及其子目录和文件
    rm -i rm -i myfile.txt 删除指定的文件时，提示用户是否确认删除
    rm -v rm -v myfile.txt 删除指定的文件，并显示详细信息 |
    cp -r cp -r /path/to/source /path/to/destination 递归地复制目录和目录中的文件
    cp -i cp -i /path/to/source /path/to/destination 在复制前提示用户是否覆盖目标文件
    cp -u cp -u /path/to/source /path/to/destination 仅复制比目标文件新或不存在的文件
    cp -v cp -v /path/to/source /path/to/destination 显示复制的详细信息
    cp -p cp -p /path/to/source /path/to/destination 保留文件的原始权限、所有权和时间戳
    cp -a cp -a /path/to/source /path/to/destination 递归地复制目录和目录中的文件，并保留文件的原始权限、所有权和时间戳
    cp --backup cp --backup=numbered /path/to/source /path/to/destination 在复制前备份目标文件，备份文件名以波浪号结尾，numbered 表示使用数字作为备份文件的后缀
    cp --force cp --force /path/to/source /path/to/destination 强制复制，即使目标文件已存在也会覆盖
